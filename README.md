#  E-Commerce SQL Analysis (UK Online Retail)

This analytics project explores real e-commerce transaction data using SQL.  
It includes raw-to-relational modeling, analytical querying, joins, subqueries, views and indexing.

##  Tech Stack Used

- Kaggle Open Dataset
